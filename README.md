# Life Expectancy Prediction - Linear Regression

This project predicts Life Expectancy using socio-economic factors.

Dataset:
Life Expectancy (WHO)

Features:
Adult Mortality
Alcohol
GDP
Schooling
HIV/AIDS

Steps:
1 Load data
2 Explore data
3 Prepare data
4 Train model
5 Evaluate model
6 Interpret coefficients
7 Plot results
8 Streamlit UI

Run:

streamlit run app.py
